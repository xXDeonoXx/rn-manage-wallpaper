# rn-manage-wallpaper

this package allows you to access wallpaper tools from native apis

## Installation

```sh
npm install rn-manage-wallpaper
```

## Usage

```js
import WallpaperManager from 'rn-manage-wallpaper';

// ...

const result = await WallpaperManager.multiply(3, 7);
```
